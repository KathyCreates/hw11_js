# Interactive Slider JS 
Репозиторій домашнього завдання №11 з JavaScript — реалізація функціоналу слайдеру за допомогою класів та прототипів.

## Зміст
- `index.html` — основна HTML‑сторінка проєкту.  
- `index‑orig.html` — початкова версія сторінки (оригінал).  
- `index‑prototype.html` — версія з реалізацією через прототипи.  
- `script.js` — JavaScript‑скрипт початкової версії.  
- `script‑prototype.js` — скрипт з реалізацією через прототипи.  
- `script‑class.js` — скрипт з реалізацією через класи.  
- `styles.css` — стилі проєкту.  
- `img/` — папка з графікою/зображеннями, які використовуються в проєкті.  
- `.gitignore` — файл для виключення небажаних файлів/папок із системи контролю версій.

## Мета проєкту
Метою цього домашнього завдання є:
- Закріпити навички створення об’єктно‑орієнтованого коду на JavaScript через класи і через прототипи.  
- Порівняти підхід з використанням класів і підхід з використанням прототипів.  
- Практикувати структуру проєкту з HTML, CSS, JS, і папкою для зображень.

## Як запустити
1. Скопіюй репозиторій на свій комп’ютер:
   git clone https://github.com/KathyCreates/hw11_js.git
   
2. Перейди в папку проєкту:
   
   cd hw11_js

3. Відкрий файл `index.html` у браузері (чи один із варіантів: `index‑prototype.html` чи `index.html`, залежно від бажаної реалізації).
4. Насолоджуйся роботою: код вже готовий до перегляду та тестування.

## Що реалізовано

* Версія через класи (`script‑class.js`) — показує сучасний підхід, використовуючи `class` синтаксис.
* Версія через прототипи (`script‑prototype.js`) — класичний підхід JavaScript до створення «класів».
* Версія початкова (`script.js`) — попередня реалізація без розділення на класи/прототипи.
* CSS‑стилі (в `styles.css`) для оформлення сторінки.
* Базова HTML‑структура (в `index.html`).
* Всі файли організовані так, щоб можна було легко порівняти два підходи.

## Вимоги

* Розуміння JavaScript: функцій, об’єктів, прототипів, класів.
